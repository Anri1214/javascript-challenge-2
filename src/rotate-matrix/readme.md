# Rotate matrix

```javascript
const arr1 = [
  [1, 2],
  [3, 4]
];

const arr2 = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9]
];

// Example 1
rotateMatrix(arr1);
// Should return

/*
[
  [1,3],
  [2,4]
]
*/

// Example 2
rotateMatrix(arr2);
// Should return

/* 
[
  [1,4,7],
  [2,5,8],
  [3,6,9]
]
*/
```
